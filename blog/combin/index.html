Is Combin Safe? Detailed and Unbiased Combin Review

November 24, 2019 in Bots, Tools

If you have been searching for the ultimate Instagram marketing tool, chances are you have heard about Combin. You’re probably wondering if this tool is worth your time. After playing around with the tool for a few days, I decided to write this Combin review to share my experience with you.

Is Combin a Bot?

Look, if you’re planning to use this tool to automatically grow your Instagram account, there’s something that you need to know. Combin is not an Instagram bot.

It doesn’t automate activity on your profile. It won’t auto follow users, like content, or leave comments. The tool is more for audience management.

In other words, a dashboard with reports, charts, and other analytical marketing tools to build an extremely targeted following.

In other words, it doesn’t perform any actions for you. But it does give you the option to manually interact with accounts.

However, I noticed Combin is planning to launch auto-tasks soon. Unfortunately, it’s unclear when the changes will be added. We will keep a close eye on Combin and update this review whenever they add new features.

Combin Products

First things first, Combin has two products: Growth and Scheduler. The Growth is focused on audience management while Scheduler is simply a tool for uploading and schedule posts and Instagram Stories.

Before you get too excited, unfortunately Scheduler hasn’t been released yet. It’s marked as “Coming Soon” which is a shame because it seems like quite a useful product.

Installation Required

Another important thing you need to know about this tool is you have to download it and run it on your computer. Luckily, there are versions for all major operating systems, such as Windows, IOS, and even Ubuntu. At the moment, there is no browser based version.

To access the program, you have to download and install it. Thankfully, the system requirements are very low, the only requirement is 4GB of RAM and 2 GB of disk space. It doesn’t consume a lot of resources, that’s for sure.

Now you’re probably wondering if you need to use a VPS to run this program 24/7. Since the tool doesn’t automate activity (it’s not a bot) you don’t need to have it open and running all the time. In other words, there’s no reason to buy a VPS for this tool.

On the other hand, if you have more than one Instagram account, Combin does recommend use proxies from their recommended list. The point is to get a dedicated proxy so your login location doesn’t keep switching every time, a clear sign you’re using a bot.

Prices & Fees

  • Starter – Free
  • Personal – $10/month
  • Business –¬† $30/month.

Note: These prices are for Combin Growth because Combin Scheduler hasn’t been released yet.

As you can see, the prices are reasonable. I like how they include a free account for you to play around with.

The Starter account does have some limits, but it should give you an idea of how the tool works. It’s also nice to know there are no time limits or hidden fees – it’s free – forever.

Eventually, Personal and Business will include auto-tasks, although we’re not sure when those features will arrive. Until then, the main difference between the subscription plans are the numbers of accounts you can connect.

With Starter and Personal, you can only connect one Instagram account. With Business, you can connect up to 5 for free, with the option to buy slots for 10 more.

Now that we got the basics out of the way, I’ll show you what this tool can do for you.

Getting Started with Combin

Think of Combin more like an Instagram user search tool. It has a lot of filters you can use to find very specific users on Instagram.

After connecting Combin to your Instagram account, the next step is to create a New Search. You can either search for posts or users.

Searching for Posts

Searching for posts is useful because you can not only like those posts but also interact with the owners of the pictures.

With Combin, you can search for posts by location or hashtag or both together. This makes it easy for you to send out “Pings” to other people in your industry.

Searching for Users

Now this feature is probably the most useful. It’s the main feature you’ll be using to build your audience. What this is mainly designed for is to build a similar audience as your competition.

To start you’ll need the username of a profile you want to compete with.

Select the “Search By” option from the drop down menu:

  1. Followers
  2. Followings
  3. Commenters
  4. Likers
  5. Bio
  6. Users List

And now you fill in the relevant results. For example, if you select the “Followers” option, the tool will find all the followers of the specific user you added earlier. Likewise, the “Commenters” option will show all the people who left comments, and so on.

You can also use the Advanced Filter and Analysis to narrow down the users even further. For instance, hide inefficient accounts, filter by gender, filter by language, filter by last activity, and what not.

If you use these filters correctly, it should be easy to build a following of active users.

Note: The Bio filter allows you to search for users that contain a specific keyword in their bio description.

Sliders

With every search, there’s a likes and comment slider, even for hashtag searches. The purpose of these sliders is to set a range of activity. It’s useful for finding active users that are using the same hashtag you’re searching for. In other words, a great way to guarantee you only find active users.

Manage Your Followers

Once cool feature of this app is you can easily manage your own followers. You can see who is not following you back and mass follow and un-follow users with a few clicks.

Remember, it’s best not to un-follow too many users at one time. Limit the amount of follows and un-follows to a handful, at least at first.

Export a List

Another great feature is the ability to export a list of Instagram users. You can make a list for almost every type of user.

Such as:

  • Users Following
  • Followers
  • Unfollowed
  • Safe List
  • Black List

This makes it very easy to organize users or import lists into other projects. Great for data geeks!

Combin Pros

Now let’s take a look at the pros.

Safe üëç

When it comes to Instagram tools, the first question people ask is, “Is It a Safe?”.

Since Combin does not automate any activity on your profile, it’s very safe. Think of it more like a Instagram browser and management tool.

On the other hand, there is the option to interact with users manually. And if you interact with too many users at a time, then Instagram might temporarily lock your account.

On top of that, you can add proxies for an extra layer of security. Overall, I feel like Combin is perfectly safe, and there’s no need to worry about being banned.

Budget-Friendly üëç

Let’s face it, Combin is a very cheap Instagram management tool. The fact that they have a free-forever starter account is great too. The paid version only costs $10 a month. Granted, it won’t automatically build your audience for you, but it’s a decent tool for searching and managing users.

Schedule Posts with Growth üëç

Growth is a separate product from Combin that can be used to upload and schedule posts. Personally, I really like this tool. Not only can you schedule posts, but also add all the relevant details, such as descriptions, hashtags, and location.

You can also schedule posts down to the minute and view all your scheduled posts in a nice grid calendar layout. Definitely worth looking into.

Easily Interact with Users and Posts üëç

One thing I really liked about this tool is you can easily interact with users and posts you find. In the search results you’ll see quick-action buttons for the actions. You can leave comments, like the post, and even direct message users.

The good news is you can also easily manage your own content, for instance, reply to comments on your posts. I thought this was really cool because you can search for hashtags using the activity sliders and then just like all the posts on there, sending out pings to lots of users.

Combin Cons

And now for the downsides.

No Auto Tasks (Coming Soon!) üëé

As I mentioned earlier, currently Combin does not include auto tasks. I see how this might be a deal breaker for people looking to quickly grow their accounts. It seems like this service has been in the planning stages for quite some time now.

Has to Run on Your Computer üëé

Unfortunately, Combin is not browser based. You have to download and install it. It can be a little bothersome to open the program and login every time you want to use it.

Case by Case Refund Policy üëé

Most software companies will grant you a full refund with no questions asked. With Combin, they’ll only refund your order if your situation meets their requirements for a refund.

They only grant refunds for specific cases. For more details on that, read their official refund policy. The good news is you cancel at any time, and it’s very cheap, so the risks are low.

Prone to Errors üëé

While researching Combin, I kept coming across posts from users struggling with various errors. It seems like Combin is prone to some unexplained errors. One of these is “Waiting for Instagram Callback” while the app straight up crashes for some users. It also seems a bit unstable, some days it works, other days not at all.

Wrapping It Up

Our overall combin review: üëé

You’re probably wondering if Combin is a tool we recommend. Although we don’t have any hard feelings towards this tool, it’s not something we wholeheartedly recommend. Nevertheless, I feel like it’s a decent tool and definitely can be used to grow a highly targeted audience.

Based on the cons mentioned earlier, I don’t think Combin is worth it for long term use. The main issue for me is the unreliable performance. I don’t want to have to sit around all day waiting for the callback error to disappear.

In addition to the errors, some times the features won’t work at all. For example, when you create a new search, no results show. It seems like recently the tool hasn’t been working well at all.
